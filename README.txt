CONTRERAS YENY
Software per il supporto del processo decisionale
Istruzioni per l'installazione e l’utilizzo dell’applicazione:
-	Effettuare il fork del progetto al seguente link URI: https://github.com/TdP-prove-finali/contreraYeny sul proprio account GitHub
-	Clonare il progetto dal link all’interno del proprio ambiente di programmazione
-	Installare le seguenti librerie : flet=0.24.1, mysql-connector-python=8.3.0, numpy=2.2.0,
-	Importare il database localmente, eseguendo il file SQL contenuto nella cartella database
-	Aggiornare i dati d’accesso al database tramite la classe DBConnect
-	Eseguire la classe Main per avviare l’applicazione
-	Nel file UI/controller.py è possibile modificare le credenziali di accesso che si desidera predisporre nella fase di login
